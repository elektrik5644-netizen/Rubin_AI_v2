ЗАПУСК СИСТЕМЫ
==============

1. Запустить C++ ядро:
   RubinEngine_Simple.exe

2. Запустить Python API:
   python start_rubin_servers.py

3. Открыть веб-интерфейс:
   RubinIDE.html

4. Или автоматически:
   start_rubin_complete_system.bat

РЕЗУЛЬТАТ
=========

Rubin IDE - это полнофункциональная система
промышленной автоматизации с AI-поддержкой,
готовая к работе с PMAC контроллерами и
технической документацией.
