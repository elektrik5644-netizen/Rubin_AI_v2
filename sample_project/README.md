
# Пример проекта для Rubin AI

Этот проект демонстрирует возможности анализа кода системой Rubin AI.

## Структура проекта

- `main.py` - основной файл с классом Calculator
- `singleton.py` - пример реализации паттерна Singleton
- `README.md` - документация проекта

## Возможности

1. **Calculator** - простой калькулятор с историей операций
2. **DatabaseConnection** - пример паттерна Singleton для подключения к БД

## Использование

```bash
python main.py
python singleton.py
```

## Паттерны проектирования

- Singleton (в singleton.py)
- Простая фабрика (Calculator)
