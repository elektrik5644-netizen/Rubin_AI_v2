# Электрические схемы и расчеты

## Основные законы

### Закон Ома
```
U = I × R
I = U / R
R = U / I
```

### Мощность
```
P = U × I = I² × R = U² / R
```

### Энергия
```
W = P × t
```

## Последовательная цепь

### Характеристики
- Ток одинаков во всех элементах
- Напряжение распределяется между элементами
- Общее сопротивление: R_total = R1 + R2 + R3

### Пример расчета
```
Дано: R1 = 10 Ом, R2 = 20 Ом, U = 12 В
Найти: ток в цепи

Решение:
R_total = R1 + R2 = 10 + 20 = 30 Ом
I = U / R_total = 12 / 30 = 0.4 А
```

## Параллельная цепь

### Характеристики
- Напряжение одинаково на всех элементах
- Ток распределяется между ветвями
- Общее сопротивление: 1/R_total = 1/R1 + 1/R2 + 1/R3

### Пример расчета
```
Дано: R1 = 10 Ом, R2 = 20 Ом, U = 12 В
Найти: общий ток

Решение:
1/R_total = 1/R1 + 1/R2 = 1/10 + 1/20 = 0.1 + 0.05 = 0.15
R_total = 1/0.15 = 6.67 Ом
I_total = U / R_total = 12 / 6.67 = 1.8 А
```

## Смешанная цепь

### Метод расчета
1. Упростить схему, объединяя последовательные и параллельные участки
2. Найти эквивалентное сопротивление
3. Вычислить общий ток
4. Определить токи и напряжения в ветвях

### Пример
```
Схема: R1(10 Ом) - R2(20 Ом) параллельно R3(30 Ом)
Напряжение: 24 В

Решение:
1. R2 и R3 параллельно: R23 = (20 × 30) / (20 + 30) = 12 Ом
2. R1 и R23 последовательно: R_total = 10 + 12 = 22 Ом
3. I_total = 24 / 22 = 1.09 А
4. U1 = 1.09 × 10 = 10.9 В
5. U23 = 1.09 × 12 = 13.1 В
6. I2 = 13.1 / 20 = 0.655 А
7. I3 = 13.1 / 30 = 0.437 А
```

## Трехфазные цепи

### Соединение звездой
- Линейное напряжение: Uл = √3 × Uф
- Линейный ток: Iл = Iф
- Мощность: P = √3 × Uл × Iл × cos(φ)

### Соединение треугольником
- Линейное напряжение: Uл = Uф
- Линейный ток: Iл = √3 × Iф
- Мощность: P = √3 × Uл × Iл × cos(φ)

### Пример расчета
```
Дано: Uл = 380 В, Iл = 10 А, cos(φ) = 0.8
Найти: мощность

Решение:
P = √3 × Uл × Iл × cos(φ) = √3 × 380 × 10 × 0.8 = 5264 Вт = 5.26 кВт
```

## Трансформаторы

### Коэффициент трансформации
```
k = U1 / U2 = N1 / N2 = I2 / I1
```

### Мощность
```
P1 = P2 (идеальный трансформатор)
P1 = U1 × I1 × cos(φ1)
P2 = U2 × I2 × cos(φ2)
```

### КПД
```
η = P2 / P1 × 100%
```

### Пример расчета
```
Дано: U1 = 220 В, U2 = 12 В, I2 = 5 А, η = 95%
Найти: I1

Решение:
P2 = U2 × I2 = 12 × 5 = 60 Вт
P1 = P2 / η = 60 / 0.95 = 63.2 Вт
I1 = P1 / U1 = 63.2 / 220 = 0.287 А
```

## Электродвигатели

### Асинхронный двигатель
- Скольжение: s = (n0 - n) / n0
- Мощность на валу: P2 = M × ω
- КПД: η = P2 / P1

### Синхронный двигатель
- Частота вращения: n = 60 × f / p
- Мощность: P = √3 × U × I × cos(φ)

### Пример расчета
```
Дано: P = 5.5 кВт, U = 380 В, I = 10 А, cos(φ) = 0.85
Найти: КПД

Решение:
P1 = √3 × U × I × cos(φ) = √3 × 380 × 10 × 0.85 = 5594 Вт
η = P2 / P1 = 5500 / 5594 = 0.983 = 98.3%
```

## Освещение

### Расчет освещенности
```
E = F × n × η / S
```

Где:
- E - освещенность (лк)
- F - световой поток лампы (лм)
- n - количество ламп
- η - коэффициент использования
- S - площадь помещения (м²)

### Пример расчета
```
Дано: S = 50 м², E = 300 лк, F = 2000 лм, η = 0.6
Найти: количество ламп

Решение:
n = E × S / (F × η) = 300 × 50 / (2000 × 0.6) = 12.5 ≈ 13 ламп
```

## Защита и автоматика

### Автоматические выключатели
- Номинальный ток: Iном
- Ток срабатывания: Iср
- Время срабатывания: tср

### УЗО
- Дифференциальный ток: IΔn
- Время срабатывания: t ≤ 0.04 с

### Реле защиты
- Ток срабатывания: Iср = k × Iном
- Время срабатывания: t = f(I/Iср)

## Кабели и провода

### Выбор сечения
```
S = I / (j × k)
```

Где:
- S - сечение (мм²)
- I - ток (А)
- j - плотность тока (А/мм²)
- k - коэффициент снижения

### Падение напряжения
```
ΔU = 2 × I × R × L / 1000
```

Где:
- ΔU - падение напряжения (В)
- I - ток (А)
- R - сопротивление 1 км кабеля (Ом/км)
- L - длина линии (м)

### Пример расчета
```
Дано: I = 20 А, L = 100 м, ΔU = 5 В
Найти: сечение кабеля

Решение:
R = ΔU × 1000 / (2 × I × L) = 5 × 1000 / (2 × 20 × 100) = 1.25 Ом/км
По таблице: S = 2.5 мм²
```

## Заземление

### Сопротивление заземления
```
Rз = ρ / (2π × L) × ln(4L / d)
```

Где:
- ρ - удельное сопротивление грунта (Ом×м)
- L - длина электрода (м)
- d - диаметр электрода (м)

### Пример расчета
```
Дано: ρ = 100 Ом×м, L = 2 м, d = 0.02 м
Найти: сопротивление заземления

Решение:
Rз = 100 / (2π × 2) × ln(4 × 2 / 0.02) = 7.96 × ln(400) = 7.96 × 5.99 = 47.7 Ом
```

## Измерения

### Мультиметр
- Напряжение: U = I × R
- Ток: I = U / R
- Сопротивление: R = U / I

### Осциллограф
- Период: T = 1 / f
- Частота: f = 1 / T
- Амплитуда: U = Uпик / √2

### Ваттметр
- Активная мощность: P = U × I × cos(φ)
- Реактивная мощность: Q = U × I × sin(φ)
- Полная мощность: S = U × I

## Безопасность

### Первая помощь при поражении током
1. Отключить напряжение
2. Вызвать скорую помощь
3. Проверить дыхание и пульс
4. При необходимости - искусственное дыхание

### Защитные меры
- Изоляция токоведущих частей
- Заземление корпусов
- УЗО и автоматические выключатели
- Средства индивидуальной защиты

### Нормы и стандарты
- ПУЭ (Правила устройства электроустановок)
- ГОСТ 12.1.038-82 (Безопасность труда)
- СНиП 23-05-95 (Естественное и искусственное освещение)
