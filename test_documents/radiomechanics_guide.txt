# Руководство по радиомеханике

## Основы радиотехники

### Электромагнитные волны
Электромагнитные волны - это колебания электрического и магнитного полей, распространяющиеся в пространстве.

#### Основные параметры:
- **Частота (f)** - количество колебаний в секунду (Гц)
- **Длина волны (λ)** - расстояние между соседними максимумами
- **Скорость распространения (c)** - скорость света ≈ 3×10⁸ м/с

#### Связь параметров:
```
λ = c / f
f = c / λ
```

### Радиочастотный спектр
| Диапазон | Частота | Длина волны | Применение |
|----------|---------|-------------|------------|
| НЧ (LF) | 30-300 кГц | 10-1 км | Навигация |
| СЧ (MF) | 300 кГц - 3 МГц | 1 км - 100 м | AM радио |
| ВЧ (HF) | 3-30 МГц | 100-10 м | КВ связь |
| ОВЧ (VHF) | 30-300 МГц | 10-1 м | FM радио, ТВ |
| УВЧ (UHF) | 300 МГц - 3 ГГц | 1 м - 10 см | Мобильная связь |
| СВЧ (SHF) | 3-30 ГГц | 10-1 см | Радар, спутники |
| КВЧ (EHF) | 30-300 ГГц | 1 см - 1 мм | Радиоастрономия |

## Антенны

### Основные параметры антенн

#### Диаграмма направленности
Графическое представление излучения антенны в пространстве.

#### Коэффициент усиления (G)
Отношение мощности излучения в направлении максимума к мощности изотропной антенны.

#### Сопротивление излучения (Rr)
Сопротивление, на котором рассеивается мощность излучения.

#### КПД антенны (η)
```
η = Rr / (Rr + Rl)
```
Где Rl - сопротивление потерь.

### Типы антенн

#### Дипольная антенна
- Простейшая антенна
- Длина = λ/2
- Сопротивление ≈ 73 Ом
- Диаграмма направленности: "восьмерка"

#### Монопольная антенна
- Вертикальная антенна над проводящей плоскостью
- Длина = λ/4
- Сопротивление ≈ 36.5 Ом
- Диаграмма направленности: "бублик"

#### Яги-антенна
- Направленная антенна
- Состоит из активного элемента, рефлектора и директоров
- Высокий коэффициент усиления
- Узкая диаграмма направленности

#### Логопериодическая антенна
- Широкополосная антенна
- Работает в диапазоне частот
- Постоянные параметры в рабочем диапазоне

#### Спиральная антенна
- Круговая поляризация
- Широкополосность
- Применение в спутниковой связи

## Модуляция

### Амплитудная модуляция (AM)
Изменение амплитуды несущего сигнала в соответствии с информационным сигналом.

#### Математическое описание:
```
s(t) = A[1 + m·cos(ωm·t)]·cos(ωc·t)
```

Где:
- A - амплитуда несущего сигнала
- m - коэффициент модуляции (0 ≤ m ≤ 1)
- ωm - частота модулирующего сигнала
- ωc - частота несущего сигнала

#### Спектр AM сигнала:
- Несущая частота: fc
- Боковые полосы: fc ± fm
- Ширина полосы: 2fm

### Частотная модуляция (FM)
Изменение частоты несущего сигнала в соответствии с информационным сигналом.

#### Математическое описание:
```
s(t) = A·cos[ωc·t + Δω·∫m(t)dt]
```

Где:
- Δω - девиация частоты
- m(t) - модулирующий сигнал

#### Преимущества FM:
- Лучшая помехоустойчивость
- Постоянная амплитуда
- Высокое качество звука

### Фазовая модуляция (PM)
Изменение фазы несущего сигнала в соответствии с информационным сигналом.

### Цифровая модуляция

#### ASK (Amplitude Shift Keying)
- Двоичная амплитудная манипуляция
- Простая реализация
- Низкая помехоустойчивость

#### FSK (Frequency Shift Keying)
- Двоичная частотная манипуляция
- Лучшая помехоустойчивость чем ASK
- Применение в модемах

#### PSK (Phase Shift Keying)
- Двоичная фазовая манипуляция
- Высокая помехоустойчивость
- Применение в цифровой связи

#### QAM (Quadrature Amplitude Modulation)
- Квадратурная амплитудная модуляция
- Высокая спектральная эффективность
- Применение в современных системах связи

## Радиоприемники

### Супергетеродинный приемник
Современная архитектура радиоприемника.

#### Принцип работы:
1. **ВЧ усилитель** - предварительное усиление
2. **Смеситель** - преобразование частоты
3. **Гетеродин** - генератор опорной частоты
4. **ПЧ усилитель** - усиление промежуточной частоты
5. **Детектор** - демодуляция
6. **НЧ усилитель** - усиление звука

#### Преимущества:
- Высокая избирательность
- Стабильность параметров
- Простота настройки

### Прямое преобразование
Альтернативная архитектура приемника.

#### Принцип работы:
1. **ВЧ усилитель**
2. **Смеситель** - прямое преобразование в НЧ
3. **НЧ усилитель**
4. **Детектор**

#### Преимущества:
- Простота схемы
- Низкое энергопотребление
- Отсутствие зеркальных каналов

## Радиопередатчики

### Основные блоки передатчика

2. **Модулятор** - наложение информации
3. **ВЧ усилитель** - усиление мощности
4. **Антенна** - излучение сигнала

### Классы усиления
#### Класс A
- Линейное усиление
- Низкий КПД (≤ 50%)
- Применение в маломощных передатчиках

#### Класс B

- Применение в двухтактных схемах

#### Класс C
- Усиление менее половины периода
- Высокий КПД (до 90%)
- Применение в мощных передатчиках

#### Класс D

- Применение в современных передатчиках

## Радиолокация

### Принцип работы радара
1. **Передача** - излучение импульса
2. **Распространение** - движение к цели
3. **Отражение** - отражение от цели
4. **Прием** - прием отраженного сигнала
5. **Обработка** - определение параметров цели

### Основные параметры радара
#### Дальность действия
```
R = (P·G²·λ²·σ) / (64π³·Pmin)
```

Где:
- P - мощность передатчика
- G - коэффициент усиления антенны
- λ - длина волны
- σ - эффективная площадь рассеяния
- Pmin - минимальная мощность приема

#### Разрешающая способность
- **По дальности**: ΔR = c·τ/2
- **По углу**: Δθ = λ/D
- **По скорости**: Δv = λ·Δf/2

### Типы радаров
#### Импульсный радар
- Излучение коротких импульсов
- Измерение времени задержки
- Высокая точность по дальности

#### Непрерывный радар
- Постоянное излучение
- Измерение доплеровского сдвига
- Высокая точность по скорости

#### Радар с синтезированной апертурой (SAR)
- Создание большой апертуры
- Высокое разрешение
- Применение в спутниковой съемке

## Спутниковая связь

### Орбиты спутников
#### Геостационарная орбита (GSO)
- Высота: 35,786 км
- Период: 24 часа
- Покрытие: 1/3 Земли
- Применение: ТВ, интернет

#### Низкая околоземная орбита (LEO)
- Высота: 160-2000 км
- Период: 1.5-2 часа
- Покрытие: глобальное
- Применение: мобильная связь

#### Средняя околоземная орбита (MEO)
- Высота: 2000-35786 км
- Период: 2-24 часа
- Покрытие: региональное
- Применение: навигация

### Параметры спутниковой связи
#### Эквивалентная изотропно излучаемая мощность (EIRP)
```
EIRP = P·G
```

#### Коэффициент качества (G/T)
```
G/T = G - 10·log(T)
```

#### Запас по мощности
```
M = EIRP - L - G/T - k - B
```

Где:
- L - потери в свободном пространстве
- k - постоянная Больцмана
- B - полоса частот

## Цифровая обработка сигналов

### Дискретизация
Преобразование аналогового сигнала в цифровой.

#### Теорема Найквиста
```
fs ≥ 2·fmax
```

Где:
- fs - частота дискретизации
- fmax - максимальная частота сигнала

### Быстрое преобразование Фурье (БПФ)
Эффективный алгоритм вычисления дискретного преобразования Фурье.

#### Применения:
- Анализ спектра
- Цифровая фильтрация
- Модуляция/демодуляция
- Сжатие данных

### Цифровые фильтры
#### КИХ-фильтры (FIR)
- Конечная импульсная характеристика
- Линейная фаза
- Стабильность

#### БИХ-фильтры (IIR)
- Бесконечная импульсная характеристика
- Высокая эффективность
- Возможная нестабильность

## Современные технологии

### MIMO (Multiple Input Multiple Output)
Использование нескольких антенн для передачи и приема.

#### Преимущества:
- Повышение пропускной способности
- Улучшение качества связи
- Повышение надежности

### OFDM (Orthogonal Frequency Division Multiplexing)
Мультиплексирование с ортогональными поднесущими.

#### Применения:
- Wi-Fi (802.11)
- LTE/5G
- Цифровое ТВ

### Массивные MIMO
Использование большого количества антенн (64+).

#### Преимущества:
- Значительное повышение пропускной способности
- Улучшение энергоэффективности
- Повышение надежности

## Практические примеры

### Расчет антенны
```python
import math

def calculate_antenna_parameters(frequency, length):
    """
    Расчет параметров дипольной антенны
    """
    c = 3e8  # Скорость света
    wavelength = c / frequency
    optimal_length = wavelength / 2
    
    print(f"Частота: {frequency/1e6:.2f} МГц")
    print(f"Длина волны: {wavelength:.2f} м")
    print(f"Оптимальная длина антенны: {optimal_length:.2f} м")
    print(f"Текущая длина: {length:.2f} м")
    
    if abs(length - optimal_length) < optimal_length * 0.1:
        print("Антенна настроена правильно")
    else:
        print("Антенна требует настройки")

# Пример использования
calculate_antenna_parameters(100e6, 1.5)  # 100 МГц, 1.5 м
```

### Расчет дальности радара
```python
def calculate_radar_range(power, gain, wavelength, rcs, min_power):
    """
    Расчет дальности действия радара
    """
    c = 3e8  # Скорость света
    pi = math.pi
    
    # Эквивалентная изотропно излучаемая мощность
    eirp = power * gain
    
    # Дальность действия
    range_max = ((eirp * wavelength**2 * rcs) / 
                 (64 * pi**3 * min_power))**(1/4)
    
    return range_max

# Пример расчета
power = 1000  # Вт
gain = 1000   # 30 дБ
wavelength = 0.1  # м (3 ГГц)
rcs = 1  # м²
min_power = 1e-12  # Вт

range_km = calculate_radar_range(power, gain, wavelength, rcs, min_power) / 1000
print(f"Дальность действия радара: {range_km:.2f} км")
```
